# Code of Conduct

## Our Pledge

We pledge to make participation in our project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

* Being respectful
* Accepting constructive criticism
* Showing empathy

## Enforcement Responsibilities

Project maintainers are responsible for enforcing the standards.

## Scope

This Code of Conduct applies within all project spaces.

## Enforcement

Instances of abusive behavior may be reported to the maintainers.

## Enforcement Guidelines

Maintainers will take appropriate action based on the severity.

## Attribution

This Code of Conduct is adapted from the Contributor Covenant.
